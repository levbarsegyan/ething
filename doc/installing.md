# Installing elioThing
## Prerequisites
- [elioWay Prerequisites](https://elioway.gitlab.io/installing.html)
## Development Setup
```shell
cd elioway
git clone https://gitlab.com/elioway/eliothing.git
cd eliothing
git clone https://gitlab.com/elioway/eliothing.git
cd eliothing
git submodule init
git submodule update
```
