## Prettier Formatting
Install `prettier`:
```bash
cd ~/repo/theElioWay
yarn
```
Run `prettier` using the `yarn format` command. This will format any code, including the MARKDOWN files in your doc folder.:
```bash
cd ~/repo/theElioWay
yarn format
git add .
git commit -m "prettier"
```
