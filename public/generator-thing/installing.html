Installing generator-thing

Prerequisites

  • elioSin Prerequisites
  • elioThing Prerequisites

Install

  • Install elioSin's yeoman generator, generator-thing globally. 
npm install -g generator-thing
yarn global add generator-thing

Development Setup

cd elioway
git clone https://gitlab.com/elioway/eliothing.git
cd eliothing
git clone https://gitlab.com/eliothing/generator-thing.git
cd generator-thing

Testing in development

cd ~/repo/elioway/eliothing/generator-thing
npm|yarn link
cd ~/repo/newtheme
npm i|yarn add generator-thing
npm|yarn link generator-thing
yo thing

Testing after development

cd ~/repo/elioway/eliothing/generator-thing
npm|yarn unlink
cd ~/repo/newtheme
npm i|yarn add generator-thing
