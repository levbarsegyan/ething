Installing with pip

In an existing python project, pip install.

pip install elio_liar

If you want to create data for an existing non-python project, install liar using the instructions below and export your random data as JSON files your can import as required in your projects.

Development Setup

.. is also a fine way to install liar using GIT as a starting project.

mkdir elioway
cd elioway
mkdir eliothing
cd eliothing
git clone https://www.gitlab.com/eliothing/liar.git
cd liar
virtualenv --python=python3 venv-liar
source venv-liar/bin/activate
pip install -r requirements/local.txt

Run the tests:

py.test -x

Run the basic script:

python liar/quickstart.py
