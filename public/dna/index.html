dna

Start reproducing the elioWay

dna converts Schema.org RDFA definitions into digestable JSON objects. This format is designed to simplify the creation of MVC models for web frameworks.

It should be noted that there are already machine readable formats of Schema available for download on their developers, for instance JSONLD format which could easily be used instead to replace this project.

dna is designed to support the elioWay and therefore to meet the goals and intentions of this project... specifically: Do everything so simply beginners can understand, use and learn from bitesized projects.
