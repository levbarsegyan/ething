dna Quickstart

Prerequisites

  • dna prerequisites met? No!

TLDR Checklist

  1. Clone the schema.org repo.
  2. Build the dna of a Thing into a readable JSON file.
dna Thing -f dna/thing.json

Just build the DNA of Schema.org classes

Install @elioway/dna globally.

npm i -g @elioway/dna
yarn global add @elioway/dna

Build a consumerable DNA file:

dna Place Person -f "mydna/mydnafile.json"

Use "mydna/mydnafile.json" as you will.

Consume dna in an MVC

Install @elioway/dna in your project or globally.

npm i @elioway/dna
yarn add @elioway/dna

Open your project.

Clone the schema.org repo into it.

git clone https://github.com/schemaorg/schemaorg.git
# or
git submodule add https://github.com/schemaorg/schemaorg.git

Call the CRISPR class to build your dna.

const CRISPR = require('crispr')
const schemaFile = `${process.cwd()}/schema/data/schema.rdfa`
const dna = new CRISPR(schemaFile)

Either:

  1. Select the Schema classes you want to use:
let contactBookDna = dna.crisprSelective(['Place', 'Person'])
  1. Grab the whole of Schema.org's classification!
let allDna = dna.crispr()

Write the dna to a file for later use:

let fileName = `${process.cwd()}/mypath/mydna.json`
dna.write(allDna, fileName)

Or consume the dna in your MVC platform.

const mongoose = require('mongoose')
const dna2MongooseSchema = require('./dna2MongooseSchema')
const Contact = new mongoose.Schema()
Contact.plugin(dna2MongooseSchema(contactBookDna))
module.exports = mongoose.model('contact', Contact)

Now write the dna2MongooseSchema!
