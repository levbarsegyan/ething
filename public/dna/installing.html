Installing with generator-thing

We provide a Yeoman generator to get you started. generator-thing scaffolds a new Django app with dna installed - all the required settings sketched out - and all the basic views, urls, and templates you need to get started are also generated.

Installing with pip

Alongside an existing Django project you can also use generator-thing - but if you prefer to manage things manually, you can pip install dna.

pip install elio-dna

Now you need to add a Django app with all the files described in the documentation.

  • Settings
  • Models
  • Views
  • Urls
  • Templates

Development

.. is also a fine way to install dna, using GIT as a starting project.

cd elioway
git clone https://gitlab.com/elioway/elioangels.git
cd elioangels
git clone https://gitlab.com/elioangels/dna.git
cd dna
virtualenv --python=python3 venv-dna
source venv-dna/bin/activate
pip install -r requirements/local.txt

Run the tests:

find . -name '*.pyc' -delete
find . -name '__pycache__' -delete
py.test -x

Publish

Activate the virtualenv.

python3 setup.py sdist bdist_wheel
twine upload dist/*
# Enter YOUR-USERNAME and YOUR-PASSWORD

Testing it:

pip install elio-dna
