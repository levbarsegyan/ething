Building a model the elioWay

genes holds a library of ready to use Django templates, built out of schema.org "Things", the elioWay.

This project has not yet been started, but the plan is this.

We will create a template for each Schema Type, tailormade for its content. We will start with the top level Things and work down. A Place template, for instance, should render it's GeoCoordinates as a Google or Leaflet map. An Event template could have a countdown clock - and so on. Each Type template may be made up of many different templates which could be reused for properties they shared.
